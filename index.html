Conectar con Supabase

Para que las fotos y videos persistan entre dispositivos, necesitas un proyecto de Supabase (gratuito en supabase.com).

Crea el proyecto, luego un bucket llamado boda-memories (modo público) y ejecuta el SQL de abajo para crear la tabla. Pega aquí tus credenciales.

▸ Ver SQL para crear la tabla 
-- En Supabase → SQL Editor, ejecuta esto:
CREATE TABLE memorias (
  id          UUID        DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at  TIMESTAMPTZ DEFAULT NOW(),
  file_path   TEXT        NOT NULL,
  nombre      TEXT        DEFAULT 'Invitado',
  mensaje     TEXT,
  tipo        TEXT        CHECK (tipo IN ('image','video'))
);
ALTER TABLE memorias ENABLE ROW LEVEL SECURITY;
CREATE POLICY "ver_todos"    ON memorias FOR SELECT USING (true);
CREATE POLICY "subir_todos"  ON memorias FOR INSERT WITH CHECK (true);

-- Storage: bucket "boda-memories" → público
-- Storage policies → INSERT para rol "anon" en boda-memories

Las credenciales se guardan en localStorage de este navegador
